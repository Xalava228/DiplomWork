ИНСТРУКЦИЯ ПО СОЗДАНИЮ FAVICON

Для создания PNG версий favicon из SVG файла "Лого.svg":

1. Откройте файл "Лого.svg" в графическом редакторе (Inkscape, Adobe Illustrator, или онлайн-конвертер)

2. Экспортируйте/сохраните в следующих размерах:
   - favicon-16x16.png (16x16 пикселей)
   - favicon-32x32.png (32x32 пикселей)
   - apple-touch-icon.png (180x180 пикселей для iOS)

3. Альтернативно, используйте онлайн-конвертеры:
   - https://realfavicongenerator.net/
   - https://favicon.io/
   - https://www.favicon-generator.org/

4. Загрузите "Лого.svg" на один из этих сервисов и скачайте готовые файлы

5. Поместите созданные PNG файлы в папку assets/

ПРИМЕЧАНИЕ: 
Современные браузеры (Chrome, Firefox, Edge, Safari) уже поддерживают SVG favicon,
поэтому сайт будет работать и с одним только SVG файлом. PNG версии нужны для
старых браузеров и некоторых мобильных устройств.
